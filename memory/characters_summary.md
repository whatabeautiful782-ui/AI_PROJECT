# Characters Summary

## 주인공 - 한태현

### 현재 역할
- 이야기의 중심 시점 인물
- 진실을 추적하는 관찰자이자, 갈등의 핵심 당사자

### 핵심 성격
- 냉정하고 계산적이다
- 쉽게 감정을 드러내지 않는다
- 하지만 가족 문제에는 이성보다 감정이 먼저 흔들린다

### 현재 욕망
- 형이 무엇을 숨기고 있는지 알고 싶다
- 형을 의심하면서도, 완전히 배신자로 확정하고 싶지는 않다
- 자신이 더 이상 이용당하지 않기를 바란다

### 현재 감정 상태
- 혼란
- 억눌린 분노
- 의심
- 버리지 못하는 애착

### 현재 관계 상태
- 형: 의심과 애착이 동시에 존재한다
- 라이벌: 경계하지만 무시할 수는 없다
- 조직: 겉으로는 따르지만 내면에서는 신뢰하지 않는다

### 행동 원칙
- 즉흥적으로 폭주하지 않는다
- 질문하고, 의심하고, 관찰하는 태도가 기본이다
- 결정적인 순간 전까지는 증거를 모으는 쪽으로 움직인다

### 말투 / 분위기
- 직접적이지만 과장되지 않는다
- 감정을 폭발시키기보다 눌러서 말한다
- 상대를 시험하듯 질문하는 방식이 많다

### 이번 구간에서 주의할 점
- 형에게 완전히 등을 돌린 상태처럼 쓰지 않는다
- 감정적이더라도 기본적으로 침착한 인물로 유지한다
- 모든 것을 꿰뚫어 보는 캐릭터처럼 만들지 않는다

## 형 - 한지훈

### 현재 역할
- 주인공이 가장 의심하는 핵심 인물
- 보호자이자 잠재적 배신자처럼 보이는 이중적 존재

### 핵심 성격
- 냉혹하다
- 통제적이다
- 감정을 겉으로 거의 드러내지 않는다
- 필요하다면 거짓말도 한다

### 현재 욕망
- 자신의 계획을 유지한다
- 주인공이 너무 빨리 진실에 접근하지 못하게 한다
- 조직 내 권력 흐름을 자신에게 유리하게 관리한다

### 현재 감정 상태
- 경계
- 압박
- 억눌린 피로
- 감추는 태도

### 현재 관계 상태
- 주인공: 의심받고 있다는 것을 어느 정도 눈치채고 있다
- 조직: 영향력을 유지하고 있지만 내부 균열도 느끼고 있다

### 행동 원칙
- 쉽게 무너지지 않는다
- 직접 설명하기보다 숨기고 통제하려 한다
- 상대가 불안해하도록 정보를 일부러 조절할 수 있다

### 말투 / 분위기
- 말이 짧고 직접적이다
- 감정 표현이 적고 건조하다
- 상대를 압박하는 대사에 강하다

### 이번 구간에서 주의할 점
- 노골적인 악역처럼 보이면 안 된다
- 주인공에게 완전히 무심한 사람처럼 그리지 않는다
- 숨기는 이유가 단순 악의만은 아닌 느낌을 유지한다

## 라이벌

### 현재 역할
- 진실에 접근한 듯 보이는 정보 제공자
- 갈등을 더 복잡하게 만드는 변수

### 핵심 성격
- 능청스럽다
- 여유롭다
- 사람을 떠보는 데 능숙하다
- 본심을 잘 숨긴다

### 현재 욕망
- 주인공을 자신의 흐름 안으로 끌어들이고 싶다
- 상황을 더 흥미롭게 만들고 싶어 한다
- 자신의 목적을 위해 정보 우위를 유지한다

### 현재 감정 상태
- 흥미
- 계산
- 여유
- 장난기

### 현재 관계 상태
- 주인공: 유용한 상대이자 흥미로운 관찰 대상
- 형: 직접 충돌하지 않지만 서로를 경계하는 관계

### 행동 원칙
- 한 번에 모든 정보를 주지 않는다
- 진실과 거짓을 섞어서 말할 수 있다
- 자신의 의도를 직접 설명하지 않는다

### 말투 / 분위기
- 장난스럽지만 가볍지만은 않다
- 가끔은 지나치게 친근하게 다가온다
- 말의 절반은 시험이고 절반은 미끼다

### 이번 구간에서 주의할 점
- 완전한 조력자처럼 쓰지 않는다
- 독자가 쉽게 믿지 못하게 유지한다
- 너무 악당처럼 단순화하지 않는다
